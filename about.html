Nutrition Assessment Data Interpreter


Nutrition Assessment and Data Intepreter is a Health mobile application which assesses Underfive Children's Nutrition status and provides immediate feedback according to the results of assessment.

Weight For Age Assessment

This is the anthropometric index which assesses immediate undernourishment occurence depending on weight and the age of the child

Height For Age Assessment

This is the anthropometric index which assesses long term undernourishment occurence depending on height and the age of the child

Weight For Height Assessment

This is the anthropometric index which assesses moderate occurence undernourishment depending on weight and height of the child

Mid Upper Arm Circumstance(MUAC)

This is the anthropometric Assessment which is done in children from 6 to 60 months to reflect their dietary intake and nourishment

Oedema Assessment

This is a clinical assessment which is done by pressing thumbs on the feet for 3 seconds and observe the outcome either pit formed or not formed. If pit formed,then the child has Oedema otherwise no Oedema. Oedema presence denotes protein deficiency(Kwashiorkor).
